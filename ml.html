Bulldozer Price Prediction

Tools:

Scikit-Learn, Kaggle Notebook Env


This project is inspired from Kaggle Competition 'Blue Book for Bulldozers'

Objective of the project is to build an ML model which predicts the sale price of bulldozers. It is a time series problem.

After basic EDA, Random Forest Regressor model from Scikit-Learn Library was built to predict prices.
