Bulldozer Price Prediction

Tools:

Pandas, Scikit-Learn, Kaggle Notebook Env


This project is inspired from Kaggle Competition 'Blue Book for Bulldozers'

Objective of the project is to build an ML model which predicts the sale price of bulldozers based on a time series data.

After basic EDA, Random Forest Regressor model from Scikit-Learn Library was built to predict prices. Hyperparameter tuning using Randomized Search CV was performed and later feature importance was studied.
